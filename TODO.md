## 技术栈建议
- **框架**：React 18 + TypeScript
- **状态管理**：Zustand（轻量）或 Redux Toolkit
- **路由**：React Router v6
- **UI库**：Ant Design
- **HTTP客户端**：Axios
- **样式**：CSS Modules + Sass
- **图表**：Mermaid.js
- **构建工具**：Vite
- **测试**：Jest + React Testing Library
